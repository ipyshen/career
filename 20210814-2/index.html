How to make image button

Method 1: Put image in the <button>

If we put the <img> between <button> and </button> directly, it will looks like this:
But this is so ugly...

Method 2: Use background image

  • Use <input type="button"> to replace <button>.
  • In CSS, apply background-image: url('path-to image'); on the <input> element.
  • Use background-size: 100%; to ensure showing the entire image.
Now looks better!

How to make href button

Use the attribute "onclick"

In the <input> element, we have an attribute called "onclick", and we can set its value to location.href='path-to-target'.
e.g. the entire <input> will looks like: <input type="button" onclick="location.href='www.github.com'">
For better looking, we can add the attribute "title" or cursor:pointer; on our <input> element.
